## 1.1.0 RECIBO VERDE (GREEN RECEIPT) STRUCTURE

**Status:** TODO - Consult Portuguese accountant

### Questions to Answer

- Should guides issue their own Recibo Verde?
- Or should LisboaPorFavor issue Recibo Verde to guides?
- What's the tax implication for each model?
- What documentation is required?
- How does commission structure affect Recibo Verde?

### Options

**Option A: Guides are independent contractors**
- Guides issue their own Recibo Verde
- Guides handle their own taxes
- LisboaPorFavor pays guides directly
- Guides responsible for compliance

**Option B: LisboaPorFavor is intermediary**
- LisboaPorFavor issues Recibo Verde to guides
- LisboaPorFavor handles tax reporting
- More complex but cleaner
- Guides less administrative burden

**Option C: Umbrella company model**
- Guides work under LisboaPorFavor umbrella
- LisboaPorFavor handles all admin
- Guides pay small fee for service
- Most professional approach

### Action Items

- [ ] Schedule consultation with Portuguese accountant
- [ ] Review each option with accountant
- [ ] Determine best model for LisboaPorFavor
- [ ] Document decision
- [ ] Communicate to guides
